# Ubuntu Utilities

## Graphics Card

AMD Monitor: 
https://github.com/clbr/radeontop

```
sudo apt install radeontop
sudo radeontop
```

Hit  `c`  to enable colors and  `q`  to quit. 
https://manpages.ubuntu.com/manpages/bionic/man1/radeontop.1.html

Benchmark: 
```
sudo apt install glmark2
glmark2
```
